Открыть сайт
Как опубликовать изменения на сайте:
1. Отредактируйте разделы и нажмите «Сохранить черновик» — проверьте через «Превью сайта».
2. Нажмите «Опубликовать» или «Скачать JSON» — замените файл data/content.json в репозитории.
3. Закоммитьте и запушьте на GitHub — сайт обновится автоматически.
